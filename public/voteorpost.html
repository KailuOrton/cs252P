About Polled: Users can Post polls on any topic they want in the format of a question with 2 answers. Users can then go ahead and vote on the existing polls. Polls can be added in realtime and accessed by refreshing the page. Stats gives the user information on what percentage of the voters have voted on a specific answer.

Future Goals for Polled!:
1.We want to implement a language modification system, that allows the user to view the website in any language they choose.
2.We want to added a timed refresh, so the user does not have to manually refresh the page to view the changes made.
3.Change the UI for the voting page to a more similar as post.
4.Change the UI for the stats page to be able to display graphical representations of the data.
